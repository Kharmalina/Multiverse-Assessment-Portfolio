
# KPK Car Dealership

A demonstration of a car dealership website incorporating authentication.


## Features

- CRUD and REST
- Login & Hashing
- Registration
- Authorization
- Admin Rights


## Installation

Install my-project with npm

```bash
  npm i
```
    
## Usage/Examples
To initialize both client and server side:
```bash
npm run dev
```
